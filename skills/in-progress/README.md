# In Progress

仍在开发中的 skills。它们还没准备好发布：预期会有粗糙边缘、破坏性变更和被放弃的实验。它们会从 plugin 和顶层 README 中排除，直到毕业进入稳定 bucket。

- **[review](./review/SKILL.md)** — 沿两个并行轴线 review 从固定点以来的变更：**Standards**（diff 是否遵循仓库的 coding standards？）和 **Spec**（diff 是否忠实实现了来源 issue/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 把文章塑造成 beats 旅程，类似 choose-your-own-adventure。选择起始 beat，只写那个 beat，然后转向下一个，直到文章自然结束。
- **[writing-fragments](./writing-fragments/SKILL.md)** — 一场 grilling session，从你那里挖掘 fragments，即异质的写作素材 nuggets，并把它们追加到单个文档中，作为未来文章的原材料。
- **[writing-shape](./writing-shape/SKILL.md)** — 接收一个原材料 markdown 文件，逐段塑造成文章，并在每一步讨论 format choices。
