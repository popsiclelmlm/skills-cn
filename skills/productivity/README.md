# Productivity

通用工作流工具，不专属于代码。

- **[caveman](./caveman/SKILL.md)** — 超压缩沟通模式。通过去掉填充内容将 token 用量减少约 75%，同时保持完整技术准确性。
- **[grill-me](./grill-me/SKILL.md)** — 围绕计划或设计进行毫不松手的访谈，直到决策树的每个分支都被解决。
- **[handoff](./handoff/SKILL.md)** — 把当前对话压缩成 handoff document，让另一个 agent 可以继续工作。
- **[write-a-skill](./write-a-skill/SKILL.md)** — 用合适结构、progressive disclosure 和 bundled resources 创建新的 skills。
